## 🚀 一、部署到 GitHub Pages（免费、无需服务器）

### 第 1 步：注册 GitHub 账号
前往 https://github.com 注册（如果已有账号跳过）。

### 第 2 步：创建仓库
1. 点击右上角 **+** → **New repository**
2. **Repository name** 填：`water-tendering`（或你喜欢的名字）
3. 选择 **Public**（公开）
4. **不要勾选** "Add a README file"
5. 点击 **Create repository**

### 第 3 步：上传文件
在仓库页面点击 **uploading an existing file**，把以下文件拖入：
```
index.html
data.json
collector.py
.github/workflows/update.yml
```
然后点 **Commit changes**。

### 第 4 步：开启 GitHub Pages
1. 仓库页面 → **Settings** → 左侧 **Pages**
2. **Source** 选择 **Deploy from a branch**
3. **Branch** 选择 `main`，文件夹选 `/ (root)`
4. 点击 **Save**

等待 1-2 分钟，页面显示 `Your site is live at https://你的用户名.github.io/water-tendering/`

**把这个链接分享给同事即可！**

---

## 🔄 二、自动更新设置

上传完成后，GitHub Actions 会**每天自动运行两次**（北京时间 8:00 和 16:00），采集最新招投标数据并更新网站。

### 手动触发更新
1. 仓库页面 → **Actions** → 左侧 **每日自动采集招投标数据**
2. 点击 **Run workflow** → **Run workflow**

---

## 🖥️ 三、本地使用

### 3.1 手动运行采集器
```bash
python collector.py
```
运行后 `data.json` 会自动更新。

### 3.2 本地预览
```bash
python -m http.server 8899 -d C:/Users/Administrator/Desktop/水利招投标平台
```
浏览器打开 `http://localhost:8899`
